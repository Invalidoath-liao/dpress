# dpress
基于多域名的博客系统

基于Halo 博客系统改造

dpress1.0 beta版本
1：支持多域名  
2：模板使用数据库存储  
3：可在线配置数据库  
4：基于dactor开发，前端交易和管理端交易都基于异步进行开发
5：兼容halo模板
6:支持一键安装，在线配置数据库地址
